### 共m个人(编号0~m-1) 第k个报数离开


上一轮 n+1 个人

该轮(n人)编号为0的在上一轮编号为 `(k)%(n+1)` (数到0~k-1，下一个就是k)

即：编号为f(n)的在上一轮编号`f(n+1)` 为 `(f(n)+k)%(n+1)`


求`f(1)=0` 时 其`f(m)`的值

```
f[1]=0;

for(int i=2;i<=m;i++){
  f[i]=(f[i-1]+k)%i;
}

```

### 变种1：编号1~m


```
f[1]=1;

for(int i=2;i<=m;i++){
  f[i]=(f[i-1]+k)%i;
  if(f[i]===0)f[i]=i;
}
result=f(m)
```

### 变种2：编号1~m,从第j个开始报数

还是和变种1一样，先算出正常报数时的最后一个人的原始编号，`result=(f(m)+j-1)%m`

```
f[1]=1;

for(int i=2;i<=m;i++){
  f[i]=(f[i-1]+k)%i;
  if(f[i]===0)f[i]=i;
}
result=(f(m)+j-1)%m
if(result===0)result=m
```
