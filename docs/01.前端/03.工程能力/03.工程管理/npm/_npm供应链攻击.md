## npm-scripts 攻击（wip）

安装第三方依赖的时候，可以触发安装后钩子（postinstall），以帮助处理一些构建后工作，比如依赖安装失败选择源码构建。

但这个钩子可能会被不法分子利用，导致安全问题。


此外，husky 使用中，推荐使用 prepare 代替 postinstall: https://github.com/typicode/husky/issues/884
