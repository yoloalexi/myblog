vw + rem：根元素配置 vw ，其他元素使用 rem

```css
html {
    font-size: 10vw;
}
button: 3rem;
```

根元素采用 vw ，解决以前 rem 布局需要额外引入脚本的问题

其他元素不用 vw ，是为了避免 PC 上最大宽度的展示问题（TODO check）
